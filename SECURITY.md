# Security Policy

Arcagrad is a self-hosted server with a real security surface Reports on
any of it are very helpful.

## Reporting a vulnerability

**Please do not open a public issue for security problems.**

Use **GitHub private vulnerability reporting**: the repository's *Security* tab
→ *Report a vulnerability*. That opens a private advisory thread.

Include what you'd want as a reporter: affected version/commit, a reproduction
or PoC, and your assessment of impact.

## Supported versions
Only the **latest release** receives security fixes.
